Extension Settings

服务器连接

配置后端服务的连接地址和认证信息。

可选；需与服务器 --token 参数一致

模型配置

配置后端服务的 LLM 模型供应商和参数。

选择供应商后自动填入，也可手动修改。

偏好设置

自定义保存行为和同步选项。

去重策略 当 Obsidian 中已存在相同链接时的处理方式。

帮助

快速参考和常见问题。

启动服务 
pnpm build
node dist/cli/index.js serve --port 8787

# 启用认证：
node dist/cli/index.js serve --port 8787 --token <secret>
快捷键

Alt + Shift + S，可在 chrome://extensions/shortcuts 自定义。
